
# Weather Forecast Task




## Features

- API source https://openweathermap.org/API
- Forms authentication
- .NET Framework 4.8


## Installation

Clone repo to Visual Studio

```bash
  Open Solution SoftWeather, Set as startup project WebApp and Run 
```
